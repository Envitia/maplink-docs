# OWSContext SDK


The [OWSContext](http://www.owscontext.org/) SDK allows a User to read,
analyse and display [OWSContext](http://www.owscontext.org/) documents
data files.

**Please contact** <support@envitia.com> **for additional information on
the implementation.**
