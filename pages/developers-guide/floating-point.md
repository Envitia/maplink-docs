# Floating Point


The MapLink libraries assume that the floating-point unit is set to that
used by the C/C++ runtime.

If you are using MapLink via .NET or similar technologies, you need to
ensure that the floating point setup matches that of the C/C++ runtime.

In addition, the DirectX Accelerator SDK requires a specific setup
(DirectX requirement).
