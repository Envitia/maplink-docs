Envitia::MapLink::TSLNArrowHead Class Reference
MapLink Pro Developer Toolkit » MapLink.Net

Detailed Description

TSLNArrowHead defines the head of an arrow.

An arrow tail must, at least, be defined with the coordinate defining the end of one arm. The end of the other arm is calculated automatically.
