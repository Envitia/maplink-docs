Detailed Description

TSLNInteropExportSet is a container class for a set of pre-processed TSLNStandardDataLayers as returned by the TSLNInteropManager.

Each item in the container consists of a TSLNStandardDataLayer and a string description of the filename that should be used when saving the data in the TSLNStandardDataLayer . The file naming conventions are based on the configuration options that were used to produce the TSLNStandardDataLayer. This filename is not compulsory but should be used when the formal naming criteria based on configuration settings is required.

The TSLNStandardDataLayers in this container are created by the data processing functions.

TSLNInteropExportSet objects may only be created indirectly via other API calls.
