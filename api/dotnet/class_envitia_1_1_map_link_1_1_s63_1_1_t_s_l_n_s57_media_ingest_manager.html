Detailed Description

This class is used to control the conversion of S57 data into a form suitable for fast display by MapLink. It is capable of both ingesting new datasets as well as managing and applying updates to existing ingested data.

During ingest callbacks will be issued to the OEM via the TSLNS57DataClient interface. For a description of which callbacks are used and when see the documentation for TSLNS57DataClient.

bool Envitia::MapLink::S63::TSLNS57MediaIngestManager::loadCatalog ( String^ catalogFile,
String^ filterConfigurationFile )

Loads the S57 dataset as specified by the given CATALOG file.This method will ingest all cells contained within the CATALOG.

During ingest callbacks will be issued via the TSLNS57DataClient. For information on when these callbacks will be invoked see the class documentation.

Parameters
catalogFileThe full path including filename to the CATALOG file for the S57 dataset.
filterConfigurationFileThe full path to the file which will be used to configure the S57 filter.
Returns
true on successful completion of an ingest, or false if a fatal error was encountered. Any errors encountered will be passed to the OEM via the notifyError() or notifyWarning() callbacks on the TSLNS57DataClient.
