Detailed Description

Specialist data layer for the handling of user drawn data.

The Custom Data Layer allows the developer to get a handle to the TSLNRenderingInterface class, thus enabling custom drawing in the drawing surface.

The developer should derive from TSLNClientCustomDataLayer in the application and implement the pure virtual draw method.
