Detailed Description

This class defines the formats that will be generated when processing a Map Link map with the TSLNOpenGLDataOptimiser.

◆ compressRasterETC1() [2/2]

void Envitia::MapLink::OpenGLSurface::TSLNOpenGLDataOptimiserFormats::compressRasterETC1 ( bool compressETC1)

Sets whether the TSLNOpenGLDataOptimiser will compress any rasters in the map into the Ericsson Texture Compression version 1 format.

If set either TSLNOpenGLDataOptimiser::rasterETCCompressionAvailable must return true or TSLNOpenGLDataOptimiserOptions::etcPackLocation() must be set to the full path to the ETCpack executable.

The use of ETCpack is subject to the terms listed in the appropriate section of the MapLink Pro 3rd Party Licences document. If these terms are incompatible with the application the ETCpack executable should be removed from the MapLink installation in order to prevent its use.
