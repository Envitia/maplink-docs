Deprecated APIs » MapLink 3D API » MapLink.InteractionModes.3D.NET

Detailed Description

This is the request handler, used for callbacks to the application.

The application should derive from this class, and pass a pointer to the constructor of the TSLN3DInteractionModeManager.
