Envitia::MapLink::Terrain::TSLNTerrainTileDataItem Class Reference
MapLink Terrain API

Detailed Description

This class is used to return data for a single tile of the terrain database.

For a TSLNTerrainDatabase, the size of the tile may vary.

The data contained within this class is the raw data from the terrain database, without any interpolation.

The application is responsible for deleting this object.

◆ getData()

array< double > ^ Envitia::MapLink::Terrain::TSLNTerrainTileDataItem::getData ( )

Query the entire data buffer.

Note: This is a 1-dimensional array, and will be (width * height) in length. The index (0,0) in the data buffer maps to the bottom-left coordinate of the tile. The index ((width-1),(height-1)) in the data buffer mas to the top-right coordinate of the tile.

Null values from the terrain data are represented by the result of getNull Value().

Returns an array, containing the tiles data.
