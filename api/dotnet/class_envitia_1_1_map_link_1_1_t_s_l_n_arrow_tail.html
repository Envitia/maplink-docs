Detailed Description

TSLNArrowTail defines the tail of an arrow.

An arrow tail must, at least, be defined with two coordinates via the setPoints() method. If no other attributes are set, the tail will default to a width of one pixel.

◆ widthGradient() [2/2]

void Envitia::MapLink::TSLNArrowTail::widthGradient ( double gradient)

Configure a width gradient. This method allows the tail to have a gradated width and defines the ratio between start and end widths. If widthGradient is not equal to 1, then startWidth (the width of the tail nearest the front head) equals width, and the end width is the result of width * widthGradient. When widthGradient = 1, endWidth = startWidth. When widthGradient < 1, endWidth < startWidth. When widthGradient > 1, endWidth > startWidth.

Parameters
gradientthe new width gradient.
