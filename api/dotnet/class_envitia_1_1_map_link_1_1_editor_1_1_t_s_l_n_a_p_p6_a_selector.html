Envitia::MapLink::Editor::TSLNAPP6ASelector Class Reference
MapLink.Editor.NET

Detailed Description

The TSLNAPP6ASelector class should be used for all interactions with the symbol selector dialog box.

The actual dialog is created in the class constructor. You can check that this has been created successfully by calling the valid method straight after construction.

Check the TSLNErrorStack::lastError() method for a description of the problem.

Envitia::MapLink::Editor::TSLNAPP6ASelector::TSLNAPP6ASelector ( TSLNAPP6ARequest^ client,
bool helpRequired,
IntPtr parent )

Default Constructor.

The first parameter should be a pointer to a TSLNAPP6ARequest derived class that will receive the update messages from the Palette.

If the helpRequired flag is true then a help button will be displayed on each page and the click action will be sent via a TSLNAPP6ARequest::onHelp() call.

The parent parameter should be used to specify a handle to the parent window.
