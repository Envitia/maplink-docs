◆ default

int Envitia::MapLink::TSLNCADRGDataLayer::FramesetZoneCollection::default[int]
get

Returns the zone number that the frameset resides in.

Returns -1 if the index or zone is invalid.

Return values are:

  • Zone H -> 0
  • Zone G -> 1
  • Zone F -> 2
  • Zone E -> 3
  • Zone D -> 4
  • Zone C -> 5
  • Zone B -> 6
  • Zone A -> 7
  • Zone 1 -> 8
  • Zone 2 -> 9
  • Zone 3 -> 10
  • Zone 4 -> 11
  • Zone 5 -> 12
  • Zone 6 -> 13
  • Zone 7 -> 14
  • Zone 8 -> 15

Polar Zones are not currently supported due to the projection required being different for other zones.
