Allows access to the Data held by an individual TSLNEntity.

◆ updateData() [1/2]

bool Envitia::MapLink::TSLNDataSet::updateData ( TSLNVariant ^ value,
int position )

Updates the variant at the specified position for the entity.

Not allowed if the TSLNDataSet is read only.

This will not allow changes to any of the Envitia specific attributes or to any external data items. Note that this method replaces the variant totally, not just updating the value. This means that the 'id' or 'key' could change.

Parameters
valuereference to the TSLNVariant object that is to update the TSLNDataSet
positionthe position of the data attribute in the TSLNDataSet to update.

Returns True if the data was successfully updated. False otherwise.
