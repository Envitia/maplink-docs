Envitia::MapLink::S52::TSLNS52FeatureLookupUtility Class Reference
MapLink.S52.NET

Detailed Description

The TSLNS52FeatureLookupUtility enables the querying of S-57 feature classes and attribute names from a supplied feature catalogue.

An example feature catalogue is "IHOS57Catalogue_v31.xml" which can be found in the config directory of a standard MapLink installation.

Catalogues can be loaded via the loadFeatureCatalogue method.
