Detailed Description

The TSLNObjectDataLayer component handles the manipulation of TSLNDynamicData Object (DDO) objects (and implicitly their associated TSLNDisplayObject (DO) objects).

By virtue of being added to a TSLNObjectDataLayer, MapLink will act as the visualisation manager for DOs associated with these DDOs.
