Detailed Description

This class provides access to set of Polylines. This is a geometry group that is drawn with the same set of rendering at the same time.

Rendering attributes can be set directly to this geometry using the inherited TSLNEntityBase.setRendering method. This is called 'Entity Rendering'.

Rendering can be applied to a large range of geometry using 'Feature Rendering' which is set by the datalayer using TSLNDataLayer.addFeatureRendering which creates a featureID. The featureID can then have attributes applied to it using TSLNDataLayer.setFeatureRendering.

A featureID can be applied to geometry on creation, or it can be added later using TSLNEntityBase.featureID.

Valid Rendering Attributes for TSLNMultiPolyline:

  • TSLNRenderingAttributeEdgeColour
  • TSLNRenderingAttributeEdgeStyle
  • TSLNRenderingAttributeEdgeThickness
  • TSLNRenderingAttributeEdgeThicknessUnits

The above is not a complete list, see TSLNRenderingAttributeInt, TSLNRenderingAttributeDouble and TSLNRenderingAttributeBool for more details.
