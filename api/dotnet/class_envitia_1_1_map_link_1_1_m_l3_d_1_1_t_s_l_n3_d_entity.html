Detailed Description

TSLN3DEntity is an interface to a generic 3D entity.

TSLN3DEntity and its derived classes are not instantiable classes. They may only be created by other calls to the API.

◆ boundingBox() [1/4]

TSLN3DBoundingBox ^ Envitia::MapLink::ML3D::TSLN3DEntity::boundingBox ( )

Query the minimum bounding box of the entity.

Since some entities, such as Symbol and Text, can vary according to the zoom factor, the extent of these entities must be calculated using their last render extent. Therefore a drawing surface through which the entity is viewed is required. This version of the boundingBox method assumes that this drawing surface has the id -1, which is the default. If the desired drawing surface has a difference id, then use the other boundingBox method that takes the drawing surface id as a parameter.

Returns the minimum bounding box of the entity.

double Envitia::MapLink::ML3D::TSLN3DEntity::distance ( TSLN3DCoord^ point)

Query the shortest distance between this entity and the specified point.

This version of the distance method uses the render extent of the entity and assumes that the render extent will be calculated from the drawing surface with the id -1.

Parameters
pointPosition to query for distance from.
Returns
the shortest distance between any points in the TSLN3DEntity and the specified position. If the entity does not contain any points then returns 0.0 and places an error in TSLNErrorStack. If the passed in point is null then -1.0 will be returned.

◆ scale() [3/4]

void Envitia::MapLink::ML3D::TSLN3DEntity::scale ( double scaleX,
double scaleY,
double scaleZ )

Scales this entity about its centroid.

For extruded entities the scaleX and scaleY are applied to the 2D shape in relation to the origin and the scaleZ is applied to the whole values of the extruded altitude and height. For non-extruded entities the scale factors are applied to the 3D position of the entity in the dimensions specified where the positive Z-axis passes from the centre of the earth through the north pole, the positive X-axis passes from the centre of the earth to the latitude of 0 and longitude of 0 and the positive Y-axis passes from the centre of the earth to the latitude 0 and longitude 90.

Parameters
scaleXScaling factor to apply in X dimension. 0 is invalid.
scaleYScaling factor to apply in Y dimension. 0 is invalid.
scaleZScaling factor to apply in Z dimension. 0 is invalid.
