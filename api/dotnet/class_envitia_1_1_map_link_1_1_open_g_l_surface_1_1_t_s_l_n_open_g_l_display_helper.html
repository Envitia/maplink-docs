Detailed Description

OpenGL display helper class contains an OpenGL drawing surface.

Note
This class is subject to change.

DESCRIPTION

This class is a background class that contains an OpenGL drawing surface. The user can get a reference to the drawing surface and modify it. The functions readPixels are used to read the data from the drawing surface in order to render his GUI control. The function createBitmap is used to create an output bitmap from the background drawing surface.

Usage Example:

// initialize the OpenGL creation parameters
TSLNOpenGLSurfaceCreationParameters^ creationOptions = gcnew TSLNOpenGLSurfaceCreationParameters();
creationOptions->useVSync(false);
creationOptions->numMultisampleSamples(0);
creationOptions->swapBuffersManually(true);
creationOptions->createCoreProfile(false);
creationOptions->enableDoubleBuffering(false);
// initialize the OpenGL Display Helper.
TSLNOpenGLDisplayHelper^ m_baseHelper = gcnew TSLNOpenGLDisplayHelper(System::IntPtr::Zero,creationOptions, false, false , true, 600, 600);
TSLNOpenGLSurface^ m_surface = m_baseHelper->drawingSurface;
// access the surface directly
m_surface->setOption(TSLNOptionEnum::TSLNOptionDoubleBuffered, true);
m_surface->reset(false);
// read pixels into readBuf array.
int totalBytes = BITMAP_WIDTH * BITMAP_HEIGHT * 4;
cli::array<System::Byte>^ _readBuf = gcnew cli::array<System::Byte>(totalBytes);
bool retVal = baseHelper->readPixels( _readBuf, totalBytes, 0);
// create image from the surface layer
m_baseHelper->createBitmap( resultImgPath );
