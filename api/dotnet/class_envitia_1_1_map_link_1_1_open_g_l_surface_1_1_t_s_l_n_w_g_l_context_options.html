Detailed Description

Note
This class is subject to change and is not complete.

This class allows the OpenGL Context options to be defined.

The OpenGL Context options are required to be as close as possible if resources are to be shared. How close is very driver dependent.

A HWND or HDC has to be specified. Failure to specify one will result in a failure to create an OpenGL context.
