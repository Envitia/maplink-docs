MapLink.S52.NET

Detailed Description

The TSLNS52UserRenderingProcedure class allows a user to perform custom drawing when mariner object rendering procedures are called in the TSLNS52DynamicRenderer.

To utilise this functionality a user should derive from this class and supply an instance to the TSLNS52DynamicRenderer upon construction.

pure virtual

This method is called by the TSLNS52DynamicRenderer when a mariner object rendering procedure is executed. Upon execution the user can use the supplied rendering interface to perform custom rendering.

Parameters
procedureNameThe 8 character name of the rendering procedure which was called.
entityThe TSLNEntity representing the S57 feature which triggered the procedure being executed.
datasetThe entity's populated dataset.
dataHandlerThe TSLNDataHandler which supplies the lookup of human-readables field names to dataset keys.
stateThe TSLNS52StateObject containing the system settings which influence the current S52 rendering.
renderingInterfaceThe dynamic renderer's rendering interface which can be used to perform custom drawing.

Return true on success, false otherwise.
