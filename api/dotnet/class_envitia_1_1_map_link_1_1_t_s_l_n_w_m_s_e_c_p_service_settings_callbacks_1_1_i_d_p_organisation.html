Envitia::MapLink::TSLNWMSECPServiceSettingsCallbacks::IDPOrganisation Class Reference

Detailed Description

This class represents the federation metadata contained 'Organization' element that should be attributed to each IDP in the federation. It is used to provide human readable information on the provider of the IDP service, potentially in various languages.

It may contain one or more organisation name, one or more organisation display name and one or more organisation URL.
