Envitia::MapLink::InteractionModes::TSLNInteractionModeRequest Class Referenceabstract
MapLink Pro Developer Toolkit » MapLink.InteractionModes.NET

Detailed Description

This is the request handler, used for callbacks to the application.

The application should derive from this class, and pass a pointer to the constructor of the TSLNInteractionModeManager.
