MapLink Pro Developer Toolkit » MapLink.Net

Detailed Description

This class represents a dimension on a layer in the WMS service metadata. It is not user-creatable but instead can either be retrieved from a TSLNWMSServiceLayer.

◆ UsesNearestValue

bool Envitia::MapLink::TSLNWMSServiceLayerDimension::UsesNearestValue
get

If true, the WMS service will use the nearest available value to the value set for this dimension when processing a request. Otherwise, only the values returned from getPossibleValue() can be set for this dimension, and specifying other values may generate a service error.
