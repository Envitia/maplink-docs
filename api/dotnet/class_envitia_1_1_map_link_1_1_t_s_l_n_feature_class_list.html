Detailed Description

Feature list accessor class.

The TSLNFeatureClassList component gives access to the feature list related to a TSLNDataLayer derived component.

The TSLNDataLayer::featureList() method returns a read-only pointer to a TSLNFeatureClassList object.

Any modifications to the featureList may invalidate the pointer, or change the order of items in the list.
