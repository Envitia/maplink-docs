Detailed Description

This is a simple interaction mode for handling the magnifying glass.

The user's application creates an instance of one these classes and adds it to a TSLNInteractionModeManager.

During mouse move events, a magnifying glass is displayed. When an RButton Up event occurs, the mode resets to the default mode.

The border around the magnifying glass may be specified using the magnifying GlassBorder method.

The size and offset of the magnifying glass may be set using the magnifying GlassProperties method.

The magnification factor may be set and queried using the magnification Factor methods. Optionally the default value may be set. The current magnification factor is reset to the default value each time the mode is activation.

double offset )

Set the size and offset properties for he magnifying glass.

Parameters
radiusThe radius of the magnifying glass, as a percentage of the drawing surface width. Thus a larger size gives a bigger magnifying glass. The default value is 15.0
offsetThe offset of the centre of magnifying glass as a multiple of the size. This a value of 0.0 makes the magnifying glass appear centred at the cursor location obscuring the map underneath and a value of 1.0 has the boundary circle of the magnifying glass at the cursor location. The default offset is 1.8
