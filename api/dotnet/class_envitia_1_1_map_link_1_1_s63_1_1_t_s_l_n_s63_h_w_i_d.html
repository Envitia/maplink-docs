MapLink.S63.NET

Detailed Description

This class represents the user's HW_ID as defined by the S63 standard. A unique HW_ID should be assigned to each individual user by the OEM.

inline

Class TSLNS63HWID.

Constructs an S63 HW_ID from the passed ASCII string.

This string should be no less than 5 characters long and only contain the characters 0-9 and A-F/a-f.

Having constructed an instance of this type, the user should query the is Valid property to ensure that the value contained is valid.
