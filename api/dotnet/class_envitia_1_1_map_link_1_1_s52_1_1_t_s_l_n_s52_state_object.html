MapLink.S52.NET

Detailed Description

S52 Dynamic Renderer requires information from the application to be able to draw many of the S57 features. This class provides the application with this capability.

The default states of the features are as follows:

Centre Symbols set to false, i.e. not dynamically shifted to the centre of S57 features. Contour Labels set to true, i.e. labels displayed on S57 contour objects. Current Date set to the date/time when the State Object was created. Current Scale set to 0. Deep Contour set 10.0. Display at all Dates set to false, i.e. objects only displayed within valid date range. Display at all Scales set to false, i.e. objects only displayed within valid scale range. Display Category Mask set to TSLNS52DisplayCategoryEnum_DisplayBase. Display Overscale Pattern set to false, i.e.the Overscale pattern is not displayed when required. Distance Tags set to 0. Full Sectors set to true, i.e. lights will be shown with full sector lines. Light Descriptions set to true, i.e. light descriptions are generated when S57 LIGHTS objects appear in view. Low Accuracy Symbols set to true, i.e. low accuracy symbols are shown on data which is deemed to be of low accuracy. Radar On set to false, i.e. Radar layer is off. Safety Contour set to 5.0. Safety Depth set to 4.0. Shallow Contour set to 2.0. Shallow Pattern set to false, i.e. the additional shallow area pattern is not displayed on areas which are deemed to be shallow. Ships Outline set to true. Show Shallow Dangers set to false, i.e. shallow dangers are not to be displayed as an underwater hazard. Show Symbols set to true, i.e. symbols are drawn. Show Text set to true, i.e. text is drawn. Simple Symbols set to false, i.e. paper chart symbols are displayed. Simple Areas set to false, i.e. paper chart area boundaries are displayed. Time Tags set to 0.0. Two Shades set to false, i.e. depth areas are displayed using four colours. Text Display set to TSLNS52DisplayText_All. The following Viewing Groups are disabled: 27040, 31010, 31040, 36020 and 36050. Zero Contour set to 0.0.
