Envitia::MapLink::TSLNCADRGFrameFileInfo Class Reference
MapLink Pro Developer Toolkit » MapLink.Net

Detailed Description

This class provides a store for basic frame file information.

It is provided only as a means of presenting the data, editing the contents of this class will not have any effect on the TSLNCADRGDataLayer it was retrieved from.
