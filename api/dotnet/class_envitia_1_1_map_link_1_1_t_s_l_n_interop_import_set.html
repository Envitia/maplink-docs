Detailed Description

TSLNInteropImportSet is a container class for a set of TSLNStandardDataLayers that are to be post-processed by the TSLNInteropManager.

Each item in the container consists of a TSLNStandardDataLayer and a filename. The filename is not currently used but is present for future extensions and to provide equivalent containers for import and export.

The TSLNStandardDataLayers in this container are created by the client and added to the TSLNInteropImportSet.
