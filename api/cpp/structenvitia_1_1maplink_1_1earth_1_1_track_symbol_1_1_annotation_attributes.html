◆ font

TSLSimpleString envitia::maplink::earth::TrackSymbol::AnnotationAttributes::font

The font to use when rendering the annotation.

The font name should be set to the filename of the required font, such as 'arial.ttf' or 'timesbd.ttf'.

The name may be a filename within the system fonts directory, or may be an absolute path to a font file.

When loading from the system fonts directory note that the available fonts will vary based on the system.

By default the font name is empty, which will result in a default font being selected.
