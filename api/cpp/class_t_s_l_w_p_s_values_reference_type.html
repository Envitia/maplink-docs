TSLWPSValuesReferenceType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents the WPS values reference type, it encodes the ValuesReferenceType element from the WPS Schema. It references an externally defined finite set of values and ranges for this input.
