Detailed Description

Represents the WPS literal ouput type, it encodes the LiteralOutputType element from the WPS Schema. It is a description of a process output that consists of a simple literal value.

The literal value is described both in terms of the data type of its set of values (e.g. integer, real, etc) and in terms of supported units of measure. Both of these properties are optional and if neither is set, the semantic implication is that the literal type is simply a string.
