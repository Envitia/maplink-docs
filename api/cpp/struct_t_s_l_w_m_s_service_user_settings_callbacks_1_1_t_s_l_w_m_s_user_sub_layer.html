TSLWMSServiceUserSettingsCallbacks::TSLWMSUserSubLayer Struct Reference

Detailed Description

This struct is to allow the user to set the sub layer's main properties[name, crs, and bounding box]. This helps when there are multiple sublayers with different extens. If we set one layer visible then reset, it will use the visible sublayer's extent.
