TSLTrackAggregatorDensity::ZoomLevel Struct Reference

Detailed Description

Definition of a zoom level used by TSLTrackAggregatorDensity

A zoom level allows different sets of aggregation configurations such as:

  • When zoomed in aggregate a set of tracks into a single symbol, showing additional detail such as position/speed
  • When zoomed out aggregate a set of tracks into a single symbol, and display a less-detailed version of the track

◆ m_triggerLevel

double TSLTrackAggregatorDensity::ZoomLevel::m_triggerLevel

The zoom level

This level is specified as Map Units Per Pixel

  • A value of zero corresponds to the map being zoomed in
  • A value of infinity corresponds to the map being zoomed out
  • This value will be compared to the (vertical) resolution of the drawing surface during aggregation
