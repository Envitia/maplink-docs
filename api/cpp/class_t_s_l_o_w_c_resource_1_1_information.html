◆ temporalExtentEnd()

int64_t TSLOWCResource::Information::temporalExtentEnd ( ) const

Returns the end of the time interval relevant to the resource, in seconds since January 1 1970 UTC.

If the resource does not define a time of interest this method returns int64_t::min.

If the resource does not define a time range of interest this method returns the same value as temporalExtentStart().

◆ temporalExtentStart()
