TSLDirectImportVectorSettings Class Reference
MapLink Pro Developer Toolkit » Direct Import API

Detailed Description

This class provides access to vector-specific settings of a TSLDirectImportDataSet

Applications may query this object via TSLDirectImportDataSet::vectorSettings.

void TSLDirectImportVectorSettings::updateFeatureClassConfig ( const TSLFeatureClassConfig * config)

Update the feature configuration for the data set

This update will be applied when the data set is next reprocessed. The application should call TSLDirectImportDataSet::reprocess before any redraw operation if this function is called.

Parameters
configThe new feature configuration to use.
