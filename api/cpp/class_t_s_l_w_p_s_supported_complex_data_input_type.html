Detailed Description

Represents a supported complex data type of a process input or output. It encodes the SupportedComplexDataInputType element from the WPS Schema.

In extension to the base (supported complex data) type, the supported complex input type is described in terms of a maximum allowable size (in megabytes).The maximum megabytes property is optional.
