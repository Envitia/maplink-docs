◆ urlLoaderCallback()

virtual bool TSLGMLURLLoaderCallback::urlLoaderCallback ( const char * url,
TSLSimpleString & localPath )
pure virtual

This method will be called whenever the TSLGMLApplicationSchemaLoader encounters a reference to an external schema via a URL. It gives the user the ability to redirect the load to a local copy of the schema file. This can help improve the performance of loading an application schema.

Parameters
urlThe URL of the schema file that is to be loaded
localPathThis can optionally be populated with an alternative, local, path to the target file
Returns
true if the provided local path should be used, false to load the original URL.
