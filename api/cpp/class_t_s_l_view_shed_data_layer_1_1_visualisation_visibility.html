Detailed Description

This visualisation will display the visibility of each point as one of 3 colours:

  • visibleColour : The point can be seen by one or more observers
  • notVisibleColour : The point can not be seen by any observers
  • noDataColour : No terrain data is available for this point

These colours may include alpha values, however full alpha blending is not supported:

  • If alpha is zero, pixels of this colour will not be displayed
  • If alpha is non-zero pixels, of this colour will be displayed

The datalayer may be made translucent using TSLDrawingSurfaceBase::setDataLayerProps.
