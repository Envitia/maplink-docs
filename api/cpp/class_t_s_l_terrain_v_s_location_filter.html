Detailed Description

A location filter, for use by the TSLTerrainViewShedAlgorithm class.

These filters will be used to determine if sample points should be processed, based on implementation-specific parameters.

The exact manner in which these filters apply depends on the specific viewshed algorithm. Some may apply them to every input point, but others may only apply them when determining the spokes used by the viewshed algorithm.

Perform this filter's operation, to determine if this point should be processed.

Parameters
xThe X coordinate of the point to filter
