Detailed Description

Specialist data layer for the handling of user drawn data.

The Custom Data Layer allows the developer to render his own data.

The developer should derive from TSL3DClientCustomDataLayer in the application and implement the pure virtual draw method.

Deprecated
Deprecated as of MapLink 11.1. Advised to use MapLink Earth SDK instead.
