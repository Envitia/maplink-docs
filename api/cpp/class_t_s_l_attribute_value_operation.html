Detailed Description

The base class for a MapLink classification operation, for use with TSLFeatureClassConfig and TSLFeatureClassifierGraduated classes. It is a virtual base class, and so cannot be used directly.

Its implementation relies on the following derived classes:

  • TSLAttributeValueOperationAnd: Takes two operations and performs a logical AND between their results.
  • TSLAttributeValueOperationElse: Provides the default case should the other operations fail.
  • TSLAttributeValueOperationOr: Takes two operations and performs a logical OR between their results .
  • TSLAttributeValueOperationNot: Takes two operations and performs a logical NOT between their results.
  • TSLAttributeValueOperationEqualTo: Is the given attribute value the same as the operation value.
  • TSLAttributeValueOperationLessThan: Is the given attribute value less than the operation value.
  • TSLAttributeValueOperationGreaterThan: Is the given attribute value greater than the operation value.
  • TSLAttributeValueOperationLessThanOrEqualTo: Is the given attribute value less than or equal to the operation value.
  • TSLAttributeValueOperationGreaterThan: Is the given attribute value greater than or equal to the operation value.

◆ operationName()

const char * TSLAttributeValueOperation::operationName ( ) const

Query the operation name for this operation.

This is a generic identifier for the operation. It is used in the TSLFeatureClassConfig::load() and TSLFeatureClassConfig::save() in the <Name> tag.

For example, the <Name> could be 'Small Pop', <PropertyName> of 'pop' and <Literal> of '50000'.

Returns
The operation name.
