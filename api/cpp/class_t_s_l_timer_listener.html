TSLTimerListener Class Referenceabstract
MapLink Time API

Detailed Description

This represents an interface for notifying users of when the state of a timer changes and also when a tick event is about to start and when the tick event has stopped.

Note: Users are responsible for managing the lifetime of listener objects.
