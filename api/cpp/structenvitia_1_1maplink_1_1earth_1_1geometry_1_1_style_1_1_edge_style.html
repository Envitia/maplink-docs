◆ stipplePattern

uint16_t envitia::maplink::earth::geometry::Style::EdgeStyle::stipplePattern

The stipple patter for the edge

Stipple patterns are specified as a bit-mask, with each bit representing a visible or invisible pixel. The length of each segment will be multiplied by stippleFactor.

For instance the following settings would produce an evenly dashed line, with each dash being 8 pixels long:

  • stippleFactor = 1.0
  • stipplePattern = 0b1111111100000000

The default stipple pattern has all bits set, in order to display a solid edge.

◆ tessellationDistance

float envitia::maplink::earth::geometry::Style::EdgeStyle::tessellationDistance

The maximum distance between points on the line, in meters. By default this is set to 75Km. If reduced this setting may improve the detail of draped geometry and stippled lines, however may introduce a performance penalty.
