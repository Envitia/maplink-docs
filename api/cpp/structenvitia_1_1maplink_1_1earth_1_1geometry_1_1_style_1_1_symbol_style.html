◆ colour

TSLRGBA envitia::maplink::earth::geometry::Style::SymbolStyle::colour

The colour to use when rendering a MapLink vector symbol. This attribute will only be used if imagePath is not set. By default the symbol colour is black.

◆ declutter

bool envitia::maplink::earth::geometry::Style::SymbolStyle::declutter

If true the Symbol will be automatically decluttered when displayed, default false.

◆ id

TSLStyleID envitia::maplink::earth::geometry::Style::SymbolStyle::id

A MapLink symbol ID, which specifies how the Symbol will be rendered. By default the symbol ID is set to 1, which will produce a simple dot.

◆ imagePath

TSLSimpleString envitia::maplink::earth::geometry::Style::SymbolStyle::imagePath

A path to an image, which will be rendered at the Symbol's location. If this path is set, and the image can be loaded it will override any symbolID configured in the style.

◆ size

unsigned int envitia::maplink::earth::geometry::Style::SymbolStyle::size

The size of the rendered symbol in pixels, by default 30.
