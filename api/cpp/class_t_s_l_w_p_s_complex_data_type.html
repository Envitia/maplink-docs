Detailed Description

Represents the WPS values complex data type, it encodes the ComplexDataType element from the WPS Schema. The object holds a combination of format, encoding, and/or schema supported by a process input or output. Additional attributes can also be assigned in the form of name and value pairs, and the data itself is also stored.

Sets the binary data held by the complex data type. If the data currently held is of string type, the content type will be updated. Note that the complex data type object will create its own copy of the data and the caller maintains ownership of the data passed in.

Parameters
valueThe binary data. This must not be 0, or the content will not be updated.
valueLengthThe size of the binary data (number of unsigned chars). This number must match with the true data size as it will be used in creating the complex data type object's own copy of the data.
