TSLOGCMIMEResponse Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » OGC API

Detailed Description

This class represents the response to a request made either to a TSLOGCService or a TSLOGCServiceManager instance. Users cannot create instances of this class, but should delete them when processed.

Member Enumeration Documentation

◆ Cacheability

enum TSLOGCMIMEResponse::Cacheability

Enumeration that describes the cacheability of the response.

Enumerator
CacheabilityUndefined 

The service or service's plugin did not define the cacheability of the response.

CacheabilityNotCacheable 

The response should not be cached. This may be because the response denotes an error.

CacheabilityTemporarilyCacheable 

The response may be cached temporarily.

CacheabilityPermanentlyCacheable 

The response may be cached permanently.
