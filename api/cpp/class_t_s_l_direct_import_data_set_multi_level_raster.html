Detailed Description

This class contains information about a multi-level raster dataset within a TSLDirectImportDataLayer.

A multi level dataset contains the base (highest resolution) of a raster, along with one or more overview levels. Each overview level represents a lower resolution of the raster, which will be used to increase processing/display performance.

For all methods which take a level index:

  • The first level (index == 0) is the lowest resolution
  • The last level (index == numLevels() - 1) is the highest resolution/base raster

Raster settings may be accessed via TSLDirectImportRaster::rasterSettings, however it is possible that the overview levels are vastly different from the base raster. One example of this is a RGB base raster with a greyscale overview. In order to support this type of data raster settings are handled in the following manner:

  • By default all settings are defined on the base raster. These may be accessed as normal via TSLDirectImportRaster::rasterSettings
  • Raster settings may be overriden on a per-level basis by calling rasterSettingsForLevel
  • The default behaviour may be restored by calling resetRasterSettingsForLevel

An instance of this object may be created by calling TSLDirectImportDataLayer::createDataSets.

void TSLDirectImportDataSetMultiLevelRaster::levelSwitchFactor ( float factor)

Set the level switch factor

This value determines when the datalayer will load a high resolution level from the data set.

  • A value of 0.0 will switch to the next level when the display scale is greater than the current level's scale.
  • A value of 0.5 will switch to the next level when the display scale is halfway between the current and next level.
  • A value of 1.0 will switch to the next level when the display scale is equal to the next level's scale.

The default value is 0.5.

The application must call TSLDirectImportDataSet::reprocess if this setting is changed.

Parameters
factorThe level switch factor to set. This value will be clamped between 0.0 and 1.0.
