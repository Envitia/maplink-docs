TSLDirectImportDataLayer::DestroyDataSets Struct Reference

Detailed Description

Deleter object for use with smart pointers

This deleter will cleanup a vector of datasets, created by TSLDirectImportDataLayer::createDataSets. Applications should use TSLDirectImportDataLayer::destroyDataSets when using raw pointers.

std::unique_ptr<TSLvector<TSLDirectImportDataSet*>, TSLDirectImportDataLayer::DestroyDataSets> dataSets( layer->createDataSets( "file.ext" ) );
