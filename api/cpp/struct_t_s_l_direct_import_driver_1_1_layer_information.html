TSLDirectImportDriver::LayerInformation Struct Reference

Detailed Description

Information about a layer within the data

A layer may either be vector or raster data. Vector layers will be processed as a group where possible. Raster layers will always be processed as separate data sets.

◆ m_overviewType

TSLDirectImportDriver::OverviewType TSLDirectImportDriver::LayerInformation::m_overviewType

The overview type of the layer. If not an overview type this will be TSLDirectImportDriver::OverviewTypeNone. Otherwise the layer is an overview of the previous non-overview layer. Multiple overview layers may be presented for a single base layer. These layers should be exposed in the following order: Layer 0 : Base raster - full resolution of the data Layer 1 : overview 1 - reduced resolution of 0 Layer 2 : overview 2 - reduced resolution of 1
