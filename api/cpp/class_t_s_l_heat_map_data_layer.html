◆ ColourScaling

enum TSLHeatMapDataLayer::ColourScaling

Defining how the colour scheme values are interpreted relative to the kernel density values.

Enumerator
NormalisedByViewExtent 

The densest spot in the current view is used to normalise the values when mapping to colour scheme.

NormalisedByWorldExtent 

Applies a consistent normalisation across the world, for a given resolution. The normalisation value is calculated once for a global resolution and then adjusted linearly when the display resolution changes.

Clamped 

Uses the specified value. Any density value larger than this will be clamped to the hottest.

◆ setSampleRadius()

void TSLHeatMapDataLayer::setSampleRadius ( unsigned char pixels)

This method is called to set the radius in pixels that a point will affect in the bitmap - larger values take longer to process. The impact of the value decreases linearly further way from the point. Default is defined by the m_defaultSampleRadius constant. 0 value is ignored.

Parameters
