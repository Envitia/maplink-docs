TSLExtrusionOperation Class Reference
MapLink Spatial Editor API » LandLink API » Miscellaneous Operations

Detailed Description

This is the TSLExtrusionOperation.

Operation Name extrusion

Additional Data Pointer to a 64-bit double number representing the number of TMC units per entered number for non-interactive extrusions. This is typically the number of TMC units per map unit. Pass 0 for an interactive extrusion where the extrusion may be placed by clicking on the map.

The operation will highlight the points of selected polylines and polygons. Users specify two points on an entity by clicking on them with the left mouse button. Clicking the right mouse button at this stage will cancel the operation. For polygons (and for non-interactive mode polylines), the user then selects which of the routes between the points is being replaced by clicking near one edge. For non-interactive mode, this click is also used to define which side of the section the parallel will be drawn.

In the interactive version of the operation, an echo is displayed, which may be dropped at a particular position, using the standard snap facilities if required.

In the non-interactive version of the operation, the user is requested to enter a distance in Map Units, as per curve-copy operation.
