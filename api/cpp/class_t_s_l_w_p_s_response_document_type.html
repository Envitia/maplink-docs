TSLWPSResponseDocumentType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents a response document, i.e. a response to an execute operation request. It encodes the ResponseDocumentType element of the WPS schema.

◆ lineage()

bool TSLWPSResponseDocumentType::lineage ( ) const

Returns true if the response document shall include lineage, false otherwise. A response document that includes lineage will include a complete copy of the data inputs and output definitions as received in the execute request.

◆ noOfOutputs()

unsigned int TSLWPSResponseDocumentType::noOfOutputs ( ) const

◆ status()

bool TSLWPSResponseDocumentType::status ( ) const

Returns true if the stored execute response document shall be updated to provide ongoing reports on the status of execution, false otherwise.

◆ storeExecuteResponse()

bool TSLWPSResponseDocumentType::storeExecuteResponse ( ) const

Returns true if the execute response document shall be stored by the server, false otherwise. Note that if this is true, then the executeResponseLocation attribute in the execute response becomes mandatory.
