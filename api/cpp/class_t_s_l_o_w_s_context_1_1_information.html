◆ timeOfInterestEnd()

int64_t TSLOWSContext::Information::timeOfInterestEnd ( ) const

Returns the end of the time interval relevant to the context document, in seconds since January 1 1970 UTC.

If the document does not define a time of interest this method returns int64_t::min.

If the document does not define a time range of interest this method returns the same value as timeOfInterestStart().

◆ timeOfInterestStart()
