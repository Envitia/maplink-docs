TSLWPSSupportedCRSsType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents the WPS supported CRSs type, it encodes the SupportedCRSsType element from the WPS Schema. It is a listing of a Coordinate Reference Systems support for this process input or output.

◆ defaultCRS() [2/2]

bool TSLWPSSupportedCRSsType::defaultCRS ( const char * newVal)

Sets the default CRS for the supported CRSs list. If CRSs are applicable to this input/output (i.e. if the input/output form is bounding box), then the default CRS is a required attribute. The process shall use this CRS unless the execute request specifies another supported CRS.

Note
newVal : The new default CRS. It is not valid to set this to 0.
Returns
true if successful, false otherwise.
