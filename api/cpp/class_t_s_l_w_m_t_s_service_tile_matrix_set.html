TSLWMTSServiceTileMatrixSet Class Reference
MapLink OGC APIs » WMTS Data Layer SDK

Detailed Description

This class represents a layer's tile matrix set link in the WMTS service metadata. It is not user-creatable but instead can either be retrieved from a TSLWMTSDataLayer, that has a service loaded, or is provided by most of the callbacks on the TSLWMTSServiceSettingsCallbacks abstract class.

Information about both the tile matrix set link (such as the matrix set limits) and the associated tile matrix set (such as the coordinate reference system and tile matrices) are accessible from this class.

To set which tile matrix set link to use, call TSLWMTSServiceLayer::select TileMatrixSet, with the tile matrix set's index as the argument.

◆ getTileMatrixAt()

const TSLWMTSServiceTileMatrix * TSLWMTSServiceTileMatrixSet::getTileMatrixAt ( int index) const

The Nth tile matrix defined by the service for this tile matrix set. Matrix set limit information is available via the tile matrix.

Parameters
indexThe tile matrix to query - the user can find the range of valid values using the numTileMatrices function

On success, this method will return the Nth tile matrix. On failure, for instance due to the index being out of range, then NULL will be returned.
