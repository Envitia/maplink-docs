TSLDirectImportRasterSettings::ChannelStatistics Struct Reference

Detailed Description

Statistics returned by TSLDirectImportRasterSettings::calculateChannelStatistics

Calculation of these statistics depends upon the TSLDirectImportDriver implementation and the contents of the raster dataset. Applications should check whether values are valid before attempting to use them.
