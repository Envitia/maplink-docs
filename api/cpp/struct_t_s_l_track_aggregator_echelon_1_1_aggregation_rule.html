TSLTrackAggregatorEchelon::AggregationRule Struct Reference

Detailed Description

A rule for aggregation by echelon

The rule will be activated if:

  • The draw resolution (MU Per Pixel) is >= m_minResolution
  • The hierarchy level of the track being aggregated is equal to m_triggerHierarchyLevel

If a rule is activated the aggregator will traverse the tree of TSLTrackGroups and select the first parent with hierarchy level == m_targetHierarchyLevel

◆ m_minResolution

double TSLTrackAggregatorEchelon::AggregationRule::m_minResolution

The minimum draw resolution required to activate the rule

This level is specified as Map Units Per Pixel

  • A value of zero corresponds to the map being zoomed in
  • A value of infinity corresponds to the map being zoomed out
  • This value will be compared to the (vertical) resolution of the drawing surface during aggregation
