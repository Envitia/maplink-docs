TSLTrackAggregatorDensity::AggregationRule Struct Reference

Detailed Description

A rule definition for density aggregation

The rule will be activated if:

  • The attribute value of a track matches the attribute value on the rule
  • If more than the specified number of matching tracks are present in a single grid cell

If this rule is triggered then any affected tracks will be replaced by a single copy of m_aggregatedTrack This track group may be displayed multiple times, but only once per cell within the aggregation grid
