Detailed Description

Represents the WPS literal input type, it encodes the LiteralInputType element from the WPS Schema. It is a description of a process input that consists of a simple literal value.

In extension to the base (literal output) type, the literal input type is described in terms of any constraints on allowable values and in terms of a default value. The default value property is optional.
