TSLWPSOutputReferenceType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents a reference to an output value that is a web accessible resource. It encodes the OutputReferenceType element from the WPS Schema.
