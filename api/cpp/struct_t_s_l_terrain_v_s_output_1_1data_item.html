TSLTerrainVSOutput::dataItem Struct Reference

Detailed Description

A single data value for the TSLTerrainVSOutput classes.

The type attribute of this object defines which data attribute should be read, in order to extract the value: typeTSLTerrainVSVisibiility - data.v typeShort - data.s typeInt - data.i typeFloat - data.f typeDouble - data.d typeBool - data.b
