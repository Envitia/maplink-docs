Detailed Description

This class provides access to each item that could possibly be processed by the Filter DataLayer.

The user does not own instances of this object. The TSLFilterDataLayer owns the instances of this object.

The constructor and destructor are private so that they are not creatable or destroyable by the user.

◆ clearGeolocation()

bool TSLFilterDataLayerDisplayItem::clearGeolocation ( )

Clears the geolocation of a display item and its attachments if the display item has had its geolocation set through a call to setGeolocation(). If the display item has not had its geolocation set, then this call will return false. When a geolocation is set or cleared then all display items on the associated data layer will require reprocessing.

Returns true on success, else false.

◆ dataLayer()

TSLDataLayer * TSLFilterDataLayerDisplayItem::dataLayer ( )

Returns NULL on failure or if the display item has not been processed yet. If the display item is invalid then the data contained on the data layer may not be valid.

◆ saveDataLayer()

bool TSLFilterDataLayerDisplayItem::saveDataLayer ( const char * filename,
TSLMapLinkVersion ver = TSL_MAPLINK_DEFAULT_VERSION,
TSLRasterType rasterFormat = TSLRasterTypeTiff,
int rasterCompressionFactor = 2 )

This method will save the data layer that the display item represents, be it a raster or vector data layer as it takes sufficient arguments to save either.

If the data layer is the special case where it contains attachments data then the attachments will also be saved to the directory that the filename specified is contained in.

Parameters
filename(Common) File name of the main data layer file and its relative or full path on disk. All associated files for the data layer will be save next to this main file.
ver(Common) The version of MapLink to save the data layer at.
rasterFormat(Raster data layer only) The format to save the rasters contained by the raster data layer in. If the filter data layer has had Raster Pyramiding enabled, then this format is ignored and the format of the raster pyramid is used.
rasterCompressionFactor(Raster data layer only) The quality of raster compression to use if the rasterFormat is either JPEG or JPEG2000. Values range from 2-255. If the filter data layer has had Raster Pyramiding enabled, then this format is ignored and the format of the raster pyramid is used.

double maxY )

Sets the geolocation of a display item and any display items to which it is the parent of. This call will only be successful if the display item in question does not have a parent display item. A user may query if a display item has a valid data defined geolocation through the 'isValid' parameter on a call to geolocation(), assuming a previous user defined geolocation has not been set. If the display item has a parent display level then this call will return false.

If this call is successful then all display items on the associated data layer will require reprocessing.

◆ setInputCoordinateSystem()

bool TSLFilterDataLayerDisplayItem::setInputCoordinateSystem ( const TSLCoordinateSystem * coordSys)

Sets a user defined input coordinate system.

If the input data contains a valid coordinate system, this will always take precedence over any user defined coordinate system.

If the input data does not contain a valid coordinate system and the user has not specified one, then the default map coordinate system will be used.

If this call is successful and the input coordinate system of the display item is changed, then all display items on the associated data layer will require reprocessing.
