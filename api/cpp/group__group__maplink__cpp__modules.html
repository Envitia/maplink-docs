MapLink Pro C++ Modules

Details the MapLink Pro C++ modules describing the library files to link to and the header files to include.

MapLink Pro is available on Windows x64, Linux x64 and Android.

Module naming conventions

Note
In the table, Name is the module name, e.g. MapLink. Paths are relative to MAPLINK_HOME.
MapLink Pro no longer ships 32-bit or Debug for Release libraries.
Build type Library file (64-bit) DLL file (64-bit)
Windows (Release) lib64/Name64.lib bin64/Name64.dll
Windows (Debug) lib64/Name64.lib bin64/Name64d.dll
Non-Windows lib64/libName.so lib64/libName.so
Note
On non-Windows:
  • The bin64 folders just contain executable programs
  • Applications should be linked against the shared libraries ('.so' files), which are located in the lib64 folder
  • The names of the libraries are the same as for Windows, except for the file extension - '.lib' and '.dll' are replaced with '.so'
