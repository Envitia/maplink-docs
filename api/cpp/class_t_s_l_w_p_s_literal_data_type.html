TSLWPSLiteralDataType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents a single literal data type that is embedded in an execute operation request or response. It encodes the LiteralDataType element from the WPS Schema.

void TSLWPSLiteralDataType::dataType ( const char *  newVal)

Sets the data type of the literal output (or input).

This data type metadata should be included for each quantity whose data type is not a string.

newVal : The data type metadata to set, for example "float", "integer" etc. The caller maintains ownership of the data type metadata, and the literal data type takes its own copy.
