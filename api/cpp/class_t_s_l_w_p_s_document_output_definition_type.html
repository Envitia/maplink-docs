TSLWPSDocumentOutputDefinitionType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents the definition of a format, encoding, schema, and unit-of-measure for an output to be returned from a process. It encodes the DocumentOutputDefinitionType element of the WPS schema.

◆ abstract()

const OWS_1_1::TSLOWSLanguageStringType * TSLWPSDocumentOutputDefinitionType::abstract ( ) const

Returns a const pointer to the abstract of the document output definition. This should be a brief narrative description of the process output. It should only be used if the abstract in the process description is not adequate. If none has been set this method returns 0.

◆ asReference()

const bool * TSLWPSDocumentOutputDefinitionType::asReference ( ) const

Returns a pointer to the asReference property of the output definition type. If none has been set this method returns 0. If this has been set to true, this specifies that the output should be stored by the process as a web-accessible resource.
