TSLWPSInputReferenceType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents a reference to an input or output value that is a web accessible resource. It encodes the InputReferenceType element of the WPS schema.

◆ body()

const char * TSLWPSInputReferenceType::body ( ) const

Returns the body of the input reference. If none has been set this method returns 0.

The body of a reference is to be used as the body of the HTTP (POST) request message to be sent in order to retrieve the referenced resource. The request is to be sent to the URL specified by href().

◆ bodyAsReference()

const char * TSLWPSInputReferenceType::bodyAsReference ( ) const

Returns the reference to the body reference. If none has been set this method returns 0.

The body as reference property specifies a remote reference to a document that holds the body of the request. See the documentation on body() for the usage of the body itself.

◆ method()

const char * TSLWPSInputReferenceType::method ( ) const

Returns the HTTP method to be used in resolving this input reference. This must either be "GET" or "POST". If none has been set this method returns 0. In which case, the default of "GET" is semantically implied.
