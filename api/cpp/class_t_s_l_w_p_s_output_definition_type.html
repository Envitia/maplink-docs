TSLWPSOutputDefinitionType Class Reference
MapLink OGC APIs » MapLink Gateway (WMS & WPS) » MapLink WPS Plugin API

Detailed Description

Represents the definition of a format, encoding, schema, and unit-of-measure for an output to be returned from a process. It encodes the OutputDefinitionType element of the WPS schema.
