Detailed Description

Represents the WPS supported UOMs type, it encodes the SupportedUOMsType element from the WPS Schema. It is a listing of a Unit Of Measure support for this process input or output.

◆ defaultUOM() [2/2]

void TSLWPSSupportedUOMsType::defaultUOM ( const TSLOWSDomainMetadataType & value)

Sets the default UOM for the supported UOMs list. If UOMs are applicable to this intput/output (i.e. if there is a list of supported UOMs), then the default UOM is a required attribute. The process shall expect input in, or produce output in this UOM unless the execute request specifies another supported UOM.

This corresponds to the 'default' element of the SupportedUOMsType element from the WPS Schema.

Parameters
valueConst reference to the new default UOM.
