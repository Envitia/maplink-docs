Feature Classes

Each coverage is divided into one or more Feature Classes. Each feature class is intended to comprise features that are closely related to one another, and all features within a given class must be of the same type. For example, political features are grouped into three feature classes, each with the prefix "polbnd" (political boundary) and with suffixes "p", "l" and "a" for point, line and area features respectively.

The table below gives a full list of the feature classes that are contained within each coverage. The feature classes provide the most convenient groupings of features for the purposes of MapLink, since the coverages are too few and the feature types too numerous to be used for this purpose.
