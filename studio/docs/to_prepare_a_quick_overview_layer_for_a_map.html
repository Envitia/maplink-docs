To Prepare a quick overview layer for a map

Define the layer, dataset and data file properties for the detailed layer of the map, and render the features using the Feature Book facilities so that the map will have the desired appearance when viewed at a detailed level.

Copy the layer with all its datasets and data files into a new layer, which will be the overview layer. Remove any data that is not relevant for this overview layer. You may also wish to change some of the configuration settings if configurable datasets are being used.

Open the Feature Book and using the Section | Clone menu option, create clones of each Feature Book Section used by the layer. These cloned Sections will be used for the Overview layer, so give them an appropriate name.

For each dataset in the Overview layer, use the Dataset | Feature Book Section property page to change the assigned Feature Book Section over to the newly created and modified clone.

Using this approach you do not have to spend time duplicating rendering settings for features that are to be rendered in the same manner in the various layers.
