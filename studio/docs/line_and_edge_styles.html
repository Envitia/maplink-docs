Line and Edge Styles

Previous versions of the MapLink Pro Studio Feature Book have possessed a feature for line rendering known as Multi Part Rendering. This was designed to apply different rendering styles to three additional components of one line: Standard, Centre and Background. This could be used for example to build up visually realistic representations of roads and railways - with a casement edge, main road colour and centreline.

MapLink Pro Studio now supports this functionality through the use of complex line styles, accessed via the usual line style selection mechanism within the Feature Book. This removes the need for additional dialogs to achieve the end result, and may be applied to polygon edges as well as lines.

In addition to Multi Part lines, MapLink Pro also has the capability to add complex line styles through extension libraries. These allow lines to be displayed in more detailed styles such as weather fronts.

When projects produced prior to MapLink Pro Studio 4.5 are read into the latest version, MapLink Pro looks at the line styles and tries to find a match within the currently available line styles. In the event of MapLink Pro being unable to find a match, it displays a dialog stating which Feature Classes had unknown line styles. The exact match information may be retrieved from the log for accurate reproduction, found via Tools | MapLink Log. You may now either modify the Feature Class to use one of the available line styles, or modify the configuration files to add the missing style. Contact Envitia support for details about modifying the configuration files.
