To add a Dataset to a Layer

  • Click on one of the layers in the Project Tree.
  • Select Layer | Add Layer or Dataset | New to access the Add Dataset dialogue.
  • Select a dataset type and a Feature Book section from the lists provided in the dialogue. If none of the existing Feature Book sections is appropriate for the new dataset, you can add a new Feature Book section (by pressing Add Section) or you can import a section from another Feature Book (by pressing Import Section).
  • Press OK to add the dataset to the layer.

The dataset will initially be created with a name of the form [ <dataset_type>], where dataset_type is the dataset type selected in the dialogue. To rename the dataset, select Dataset | Rename

Dataset Coordinate System Information

For each dataset that you add, choose the appropriate one of the following instructions:

  1. If the data is in lat/long and using the WGS-84 datum. No input coordinate system information should be entered for the dataset. MapLink Pro Studio will then assume that the data is in Lat/Long and using the WGS-84 datum.
  2. If the data in the dataset is already in the coordinate system required for the output map, rhis information should be entered into MapLink Pro Studio:
  • Select Dataset | Input Coordinate System
  • Select the Advanced tab
  • Activate the Data in datset is already in Output Coordinate System box
  • If the data is projected, but not into the required coordinate system for the output map Information about the coordinate system of the dataset must be entered into MapLink. Either:
  • Apply a pre-defined coordinate system to the dataset. See:

Selecting a pre-defined Coordinate System

Or:

  • Apply and edit a coordinate system for the dataset. See:

Viewing/ Editing the current Coordinate System

Or:

  • Create a new user-defined coordinate system to apply to the dataset. See:

Creating a new user-defined Coordinate System

Moving, Copying and Deleting Datasets

The ordering of datasets within the layer determines the order in which the datasets will be rendered. Datasets at the top of the list within the layer will be rendered first, and therefore will be overlaid with datasets lower down in the list.

  • To re-order the datasets within a layer, simply click on a dataset and drag it to its new position within the layer.
  • You can also freely move a dataset from one layer to another by dragging it into the appropriate position in the new layer.
  • To copy a dataset, hold down the Ctrl key whilst dragging the dataset. You can copy to a new layer or within the same layer.
  • To delete a dataset, right-click on the dataset and select Delete from the popup menu. Confirm the deletion to remove the dataset from the Project Tree.
  • You can also move, copy and delete datasets using the Cut/Paste, Copy/Paste and Delete options on the Edit menu.
