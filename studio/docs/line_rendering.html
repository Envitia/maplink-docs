Line Rendering

Line Features can have the following properties to define their visualisation:

  • Style. This may be set using the Style Combo Box. See Line and Edge Styles for further information.
  • Colour. This may be set using the Colour Palette. See Colour Palettes for further information.
  • Thickness. This may be set using the Numeric Value Text Box.
  • Thickness Units. This may be set using the Dimension Units Combo Box. Note : changing this value will reset the Thickness to 1.

The value chosen for Thickness Units defines how the Thickness property is interpreted. Lines that have Thickness Units of Map Units, will get wider as the user zooms in. The calculated thickness is clamped so that it will always not disappear, nor be larger than 250 pixels.

Unlike earlier releases of MapLink Pro, these settings will not affect the rendering of Polygon Features. Polygons have their own settings for Edge Rendering Properties.
