MapLink Pro Map Viewer

The MapLink Pro Map Viewer allows you to see how your maps will appear within your application. Once you have created your output map, select Tools | Map Viewer to run the application.

The MapLink Pro Map Viewer is a stand-alone application and can also be run from the Start menu without first having to run MapLink Pro Studio.

See the MapLink Pro Map Viewer Help for further information.
