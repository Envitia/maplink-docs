User-Defined Colours

In order to add, edit or delete user-defined colours, it is necessary that the Colour Palette dialog is pinned. The Add, Edit and Delete buttons are not available otherwise.

To add a user-defined colour

  • Press Add… to access the 24-bit palette.
  • Define the new colour in any of the three following ways:
  • Click on the main colour area (which is a map of hue against saturation) and adjust the luminosity slider.
  • Enter the colour using RGB values in the range 0-255.
  • Enter the colour using Hue, Saturation and Luminosity values in the range 0-255.
  • Press OK to add the colour to the palette.

The user-defined colours are saved in a palette that is stored within the Feature Book. This allows them to be imported into other projects.

To edit a user-defined colour

  • Select the user-defined colour that is to be edited.
  • Press Edit… to access the 24-bit palette.
  • Change the colour as desired in any of the ways described above.
  • Press OK to modify the colour in the palette.

To delete a user-defined colour

  • Select the user-defined colour that is to be edited.
  • Press Delete
  • Confirm the deletion to remove the colour from the palette.
