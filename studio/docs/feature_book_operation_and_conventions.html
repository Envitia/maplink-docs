Feature Book Operation and Conventions

Open the Feature Book by means of the Feature Book button of the toolbar on the left of the MapLink Pro Studio main screen, via the Tools | Feature Book menu option, or by double clicking on a map feature with Select Mode enabled.

The Feature Book contains the Feature Tree and can be browsed much like standard Windows file browsers. Each Feature Class entry in the tree displays sample rendering for the four primitive types that can be produced. A secondary Feature Properties dialog is used to modify the rendering properties for the selected Feature Classes. Multiple Feature Classes can be selected and any changes made in the Feature Properties dialog will be applied to the entire selection. When no editable Feature Class is selected, the Feature Properties dialog is greyed out and the user cannot interact with it.

Changes in the Feature Properties dialog are applied immediately thus giving instant feedback. They can be reversed with the Undo button. Changes are saved when the project is saved, the Feature Book closed or the Book | Save menu item is selected. At any point, the entire Feature Book may be reverted to its initial or last saved state by selecting Book | Revert.

The Feature Properties dialog can be closed independently of the Feature Book, but closing the Feature Book will close the Feature Properties dialog as well.

N.B. The Feature Book must be closed when adding data or a dataset to a project.
