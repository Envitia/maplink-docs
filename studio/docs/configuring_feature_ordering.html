Configuring Feature Ordering

To ensure that Polygons and Polylines of Feature Class A are drawn before Polygons and Polylines of Feature Class B:

  • Open the Feature Book
  • Select Feature Class B
  • If necessary select Feature | Properties to open the Feature Properties dialog.
  • In the Feature Properties dialog, set the Render Level to be greater than that of Feature Class B.
  • NOTE: Render Levels should only be used when other Feature Ordering facilities are insufficient.

To ensure that file A is drawn before file B:

  • Select file A in the project tree
  • Drag file A to a position above file B in the tree
  • It is not necessary to reload the layer to see these changes.

To ensure that Feature Class A is drawn before Feature Class B when they are found in the same file:

  • Open the Feature Book
  • Select Feature Class A

Additionally the Feature Book has the following tools available on the toolbar:

  • Undo Available when Feature Book has been modified
  • Redo Available when Undo has been applied to redo the change.
  • Drag Feature Class A above Feature Class B in the Feature Tree with the left mouse button. Note that a Feature Class can only be moved within the current parent Feature. It is not possible to move Feature Classes to a different parent.
  • It is not necessary to reload the layer to see these changes.
  • Note that it is possible to select multiple Features from within the same parent Feature Class and move them together.
