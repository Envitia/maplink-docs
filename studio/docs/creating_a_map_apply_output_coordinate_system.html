Apply an Output Coordinate System

The output coordinate system is the coordinate system required for the finished map.

To define the output coordinate system:

  • Select Map | Output Coordinate System
  • Either:
  • Apply a pre-defined coordinate system to the map. See:

Selecting a pre-defined Coordinate System

Or:

  • Apply and then edit a coordinate system for the map. See:

Viewing/ Editing the current Coordinate System

Or:

  • Create a new user-defined coordinate system to apply to the map. See:

Creating a new user-defined Coordinate System
