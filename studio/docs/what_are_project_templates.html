What are Project Templates?

All new MapLink Pro Studio projects are based on a project template. These templates define common settings that are inherited in the new project. This means that when creating several different projects that share a number of common settings, such as the output map coordinate system, layer structures or Feature Book entries, the desired values only need to be configured once on a project template as all subsequent projects that use the template will share all of these settings.

The default template for MapLink Pro Studio can be used to set a default map output version (corresponds to a MapLink Pro Version). This is however limited in that to be able to set the map version you need to save the template project as a MapLink Pro 6 or newer version Studio project.

The default MapLink map version can be set via 'Project->Properties' from within Studio. To change the version load the default_project.mlp template (found in the directory projects\Default Template) and select the version required. Save the template to ensure the chosen version persists (save as MapLink version 6 or newer). You may need to change the file permissions to be able to overwrite the default template files.

You are still able to select a different output version on the Save dialog.
