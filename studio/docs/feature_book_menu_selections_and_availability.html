Feature Book Menu Selections and Availability

The Section menu contains the following options:

  • Import Always available for selection
  • Clone Always available for selection
  • Delete Available when an unused Section is selected

The Feature menu contains the following options:

  • Copy Available Feature Class is selected
  • Paste Available when Copy has been used
  • Delete Available when an unused Class is selected
  • Properties Available when a Feature Class is selected

The Palette menu contains the following options:

  • Import Always available for selection
  • Save the current palette file

Additionally the Feature Book has the following tools available on the toolbar:

  • Undo Available when Feature Book has been modified
  • Redo Available when Undo has been applied to redo the change.
