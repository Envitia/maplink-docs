Welcome to MapLink Pro

Welcome to MapLink Pro from Envitia, the leading software for building high performance operational mapping and spatial information applications. MapLink Pro provides an extensive range of facilities to help you work with a wide range of map data and build fast, flexible, mission-critical applications. MapLink Pro drastically cuts the time needed to develop sophisticated mapping applications - you can be up and running within minutes!

Extensive APIs

MapLink Pro supports developers using Windows 7, Windows 8.0, Windows 8.1, Windows 10, Windows Server 2012, Windows Server 2012R2, Linux and Android (contact Sales for a full list for supported platforms and to discuss custom ports of MapLink Pro) with a comprehensive set of APIs.

If you are using Microsoft Visual C++, you will be able to create a fully featured mapping application in seconds using the supplied MapLink Pro AppWizard. You can build and run the Visual C++ project directly to give you a MapLinkPro run-time application supporting zoom and pan operations for any map generated by MapLink Pro Studio.

Full instructions for this task can be found in the MapLink Pro Developer's Guide, accessible from the start menu.

High performance maps

MapLink Pro Studio allows you to build the maps to be used within the run-time applications you build using the MapLink APIs. Please take the time to learn how to make best use of the wide range of facilities provided by MapLink Pro Studio; they are the keys to creating effective, high performance maps. Once you have mastered the basic processes, check out the tips for fast maps, to learn how to get the ultimate in performance from your maps.

The MapLink Map Viewer lets you see maps you create using MapLink Pro Studio as they will appear in a run-time application. You can invoke the MapLink Map Viewer from within MapLink Pro Studio or separately from the Start menu.
