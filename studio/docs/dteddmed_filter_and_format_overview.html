DTED Filters and Format Overview

The DTED (Digital Terrain Elevation Data) filters allows MapLink Pro Studio to read data supplied in DTED format. In general, DTED data is provided in discrete .dt0, .dt1, .dt2, or .dted files at an elevation post spacing of 30 arc seconds. Alongside the DTED data files is a DMED (Digital Mean Elevation File) file. This contains data covering a 15 by 15 minute area in a one degree cell. The process described below is exactly the same for DTED and DMED files.

Each DTED file defines as a set of spot elevation values, arranged over a rectangular grid. MapLink Pro Studio provides the facility to convert these grids into either a set of annotated contour lines, or into a shaded relief raster, using either the DTED or DTEDRASTER dataset types.

To create annotated contour lines

  • Create a dataset of type DTED
  • Add the individual DTED files to the dataset
  • Use the DTED Configuration Panel to define the contour levels you require
  • Load the layer

To create a shaded relief raster

  • Create a dataset of type DTEDRASTER
  • Add the individual DTED files to the dataset
  • Use the DTED Configuration Panel to define the shading levels you require
  • Load the layer
