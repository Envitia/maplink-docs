Create a Project

The MapLink Pro Studio Project file (with extension .MLP) is the starting point for all work with MapLink Pro Studio. The map data you use within the project may be held anywhere on your computer or on the network available to your computer; the Project file will manage the links for you.

The Project file also contains Map, Layer, Dataset and Data File definitions and configuration information and provides the link to the Feature Book used within the project.

To create a new Project

  • Select Project | New to access the New dialogue.
  • Select the template project that you wish to use for the project. All new projects are based on a project template. By default, the a project template that you last worked with will be suggested, but you may browse around your computer or network for another project template. If you wish to avoid using a project template on which you have previously worked, the default project template (default_project.mlp) provided with MapLink Pro Studio may be found in the /Projects/Default Template directory of your MapLink Pro Studio installation.
  • Choose whether you wish to extend the selected project templateâ€™s Feature Book, or whether you wish to make a copy of its Feature Book. In the latter case, all changes to the Feature Book will be saved to a new Feature Book file that will be created in the directory where the new project file is saved.
  • Press OK to accept

All projects have a Map level, so a new empty project is created with a Map (named Map0 in the default project template) serving as the root of the Project Tree. The name of the Map in the Project Tree will be used as the default map name when creating the output map; it can be changed using Map | Rename.
