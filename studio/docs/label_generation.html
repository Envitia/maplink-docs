Label Generation

Label generation allows the user to select a line feature, to merge all lines of that feature code with a common attribute and to generate text entities along the merged line. This is useful for placing labels along road networks where the source data does not provide explicit text features.

The label generation panel allows the following operations

Add: Brings up the label definition wizard to guide you through the process of creating a new label definition.

Edit: Brings up the label definition wizard allowing you to edit the settings of the selected label definition.

Delete: Removes the selected definition from the label definition list.
