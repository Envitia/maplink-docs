Add Layers

To add a Layer to the Project

  • Click on the Map level in the Project Tree.
  • Select Map | Add Layer or Layer | New to add the layer to the project.

The layer will initially be created with a name of the form layer <n>, where n=1,2,3... depending upon which layers have already been created. To rename the layer, select Layer | Rename...

Layer properties can be defined at this stage, but in general it is best to load data into the layer before defining these properties.

Moving, Copying and Deleting Layers

The ordering of layers within the Project Tree has no significance, although most users will place the least detailed layers at the top of the tree and the most detailed layers at the bottom.

  • To re-order the layers within the Project Tree, simply click on a layer and drag it to its new position within the Project Tree.
  • To copy a layer, hold down the Ctrl key whilst dragging the layer.
  • To delete a layer, right-click on the layer and select Delete from the popup menu. Confirm the deletion to remove the layer from the Project Tree.
  • You can also move, copy and delete layers using the Cut/Paste, Copy/Paste and Delete options on the Edit menu.
