ArcGrid Overview
ArcGrid Filter Overview

ArcGrid is a gridded format developed by ESRI (no official specification has been published). The grid points are referred to as cells and contain a numeric value which normally represents a height value (this is the assumed meaning for Studio).

There are two main formats:

  • ASCII
  • Binary

Studio can read all known variations of the ASCII format and the majority of the Binary formats and the common Float format.

We would recommend the use of the Binary format as the floating point representations are stored as IEEE floating point. The ASCII version will suffer from floating point representation conversion limitations. This is important for the calculation of the grid extent and cell positioning.

If you use the ASCII version we would recommend that you have a one column and row overlap. This is similar to the DTED/DMED approach.

Note: OS Land-Form Panorama data can be read by this filter.
