File Geodatabase (FileGDB)

ESRI File Geodatabase - a vector format that can store a mixture of features in a directory. OGR can read file geodatabases created by ArcGIS 10.0 and later.

Support for this format is provided via OGR.

The end user has to install the DLLs obtained from ESRI as Envitia is not permitted to ship the DLLs with MapLink Pro Studio.Installation instructions are here.

To load the data select the directory containing the File Geodatabase.
