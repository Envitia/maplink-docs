Map Coordinate System Information

To define the coordinate system required for te output map:

  • Select Map | Output Coordinate System

This takes you to the Map Properties/Coordinate System page.

From this page you can access:

  • The Coordinate System Browser - This enables you to view details of all the available coordinate systems and to select one for the map.
  • The Coordinate System Viewer - This enables you to view or edit the currently selected coordinate system for the map
  • The Coordinate System Wizard - This enables you to create a new user-defined coordinate system.

For information about the following operations, please see the appropriate sections:

Selecting a pre-defined Coordinate System
Creating a new user-defined Coordinate System
