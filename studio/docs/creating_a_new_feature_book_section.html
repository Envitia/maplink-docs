Creating a New Feature Book Section

Adding a new feature book section can be achieved by one of four methods:

  1. When a new Dataset is added using the Add Dataset dialog
    • Highlight the layer in the project tree and either
      • Right click on the layer and select Add Dataset,
      • Select Layer | Add Dataset from the menubar or
      • Press the Add Dataset button from the Processing toolbar.
    • Choose the data type to be read in from the Dataset Type section.
    • Press Add from underneath the Feature Book Section - the 'New Section' dialog will be displayed.
    • Enter a unique name for the new section (you will be prompted if it conflicts with one already in existence) and press OK
    • Press OK on the Add Dataset dialog.
    • Opening the Feature Book will show the new Section ready for classes to be added by loading data into the dataset.
  2. Cloning current Feature Book sections from within the Feature Book
    • Select Section | Clone.
    • From the resultant 'Paste Feature Book Section' dialog choose the section that you would like to clone and enter a unique name in the New Name section at the bottom of the dialog. (You will be prompted if the name is not unique).
    • Press OK. The new entry is shown in the Feature Book tree, with the exact attributes of the section copied, and is available for edit or deletion of settings.
  3. From the Dataset Properties dialog
    • Highlight the dataset in the project tree and either right click and select Properties or select Dataset | Properties from the menubar.
    • Select the Feature Book Section tab, followed by Feature Book - Add Section
    • Type the new section's name in the dialog box offered
    • Press OK.
    • The next time that the Feature Book is opened the new section will be available for edit.
  4. Import a new section into the Feature Book
    • On the main Feature Book Dialog, select Section | Import and use the file browser to find the Feature Book that you wish to copy the section from
    • Press Open
    • Select the Feature Book section that you wish to import from the list in the Import Feature Book Section dialog
    • Enter a unique name for the section in the New Name dialog
    • Press OK. You will be prompted if section name duplication occurs.

The Imported Feature Book Section is displayed in the Feature Book Tree with the exact attributes of the section copied and is available for edit or deletion of settings.
