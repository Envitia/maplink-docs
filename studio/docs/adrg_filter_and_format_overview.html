ADRG Filter and Format Overview

ADRG (Arc Digitized Raster Graphic) is the predecessor of CADRG and is obsolete.

However MapLink Pro supports it as much existing data is still provided in ADRG.

ADRG requires no specific configuration and so no Format Configuration panel is present for this filter.
