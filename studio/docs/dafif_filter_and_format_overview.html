DAFIF Filter and Format Overview

Digital Aeronautical Flight Information File (DAFIF) vector data provides key air information e.g. air routes, runways, special use airspace. DAFIF is supplied in two formats: fixed records and tabbed ASCII format. MapLink supports the tabbed format. Regular data updates are provided.

DAFIF is inherently a dataset containing logical data rather than data purely for presentation. MapLink Pro Studio allows much of the data to be visualised and also allows the application to interrogate the visualisation. Significant attribute information can be displayed and interrogated.

The following items can be visualised spatially

  • Airport
  • Navaid
  • Air Refueling Point
  • Air Refueling Segment
  • Air Traffic Control Routes
  • Airport ILS
  • Airport Runway
  • Airport Runway Centre
  • Boundary Segment
  • Heliport
  • Heliport Helipad
  • Military Training Route Segment
  • Off route Terrain Clearance Altitude
  • Parachute Jump Area Segment
  • Preferred Route Segment
  • Special Use Airspace Segment
  • Waypoint

Subclassing of the DAFIF data is available via the DAFIF Format Configuration panel.
