DBDBV Filter and Overview

DBDB-V, Digital Bathymetric Database, is a US standard that is being widely adopted as the Bathymetric equivalent of DTED, Digital Terrain Elevation Database. Like DTED it is a global gridded dataset, with higher resolution data available in different areas. The V indicates variable spacing. Within a DBDB-V dataset there is also the capability to support different resolutions of data. DBDB-V version 5 supports increased resolution over previous versions, allowing data down to one meter grid spacing.

MapLink Pro Studio supports level 1.0 and level 4.0 DBDB-V as well as DBDB-V version 5 (5.4) and DBDB-V version 6 (6.2) via different dataset types.

Level 1.0 and 4.0 data can be displayed as a shaded raster using the DBDBV-RASTER dataset, as a set of vector contours via the DBDBV-VECTOR dataset and can also be exported as a terrain database via the DBDBV-TERRAIN dataset.

Version 5 data can be used in all of the above ways using the DBDBV-V5-RASTER, DBDBV-V5-VECTOR and DBDBV-V5-TERRAIN datasets.
