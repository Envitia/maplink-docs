OpenStreetMap

OpenStreetMap data is a collaborative, free, worldwide, street-level set of map data.

The main download of OpenStreetMap data is huge. It is therefore recommended that you download or extract only the data for the area of interest.

There are three mechanisms documented:

  1. Using the export tab on the OpenStreetMap Viewer
  2. Using one of the API map requests
  3. Extract the area of interest from the downloaded data using osmosis

See the following: http://wiki.openstreetmap.org/wiki/Downloading_data

Example use of osmosis:

  • osmosis --read-xml enableDateParsing=no file=..\Data\OSM\great-britain-latest.osm --bounding-box top=51.126 bottom=50.729 left=-0.03 right=0.7 --write-xml file=new.osm
