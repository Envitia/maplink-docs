GDF Configuration Panel

The GDF Configuration Panel allows the user to access attribute information provided as an inherent part of GDF data. This attribute information can be used to subclass individual native features in a more meaningful way.

This panel displays the available native feature codes in human readable form if the source GDF file provides a feature definition block. If there is no human readable form the feature code number will be displayed. As each native feature code is selected a list of available fields will be displayed in the Available Fields list. This list is read only and selecting items will have no effect.

The selected native feature may be sub-classed by two levels in the Feature Code Configuration Panel.

To subclass features for a native feature

  • In the panel, select the field that is to provide the first level of sub-classing from the Level 1 list
  • Tick the Subclass Features from Theme check box
  • If a second level of sub-classing is required, select the field that is to provide it from the Level 2 list
  • Press OK
  • The Data or Dataset affected will be re-catalogued to obtain the new feature code set according to the new settings

The panel also allows the user to select the language to use when extracting appropriate textual information. Note if there is only one alternative available for an attribute name, then this will be provided irrespective of selected language.

The user may also elect to give each of the line features a full set of attributes if they are to be used later for line merging or entity naming. It is recommended that the full set of attributes is not collected for each feature in order to reduce memory usage. The standard sets of attributes are given below.

Address Area

For the road elements, the major line feature of interest, the standard set of attributes provides:

  • "ON" "AN" "BP" "FW" "SL" "FY" "FC" "RL","RN".

For road junctions the standard set of attributes provides

  • "JT"

For ferries the standard set of attributes provides

  • "ON", "AN" , "FT" , "FC" , "RL"

All other line features provide a full set of attributes.
