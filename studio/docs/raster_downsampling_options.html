Raster Downsampling Options

When working with Raster Data, if you have used Raster Downsampling to create optimal performance you may wish to store the pyramid layer structure in a single file. This large file, and its associated control file, will contain the individual layers as well as header information describing how the layers fit together. The advantage of this option is that keeps the data together while neatly controlling the memory usage.

The size of the pyramid nugget may be entered or you may select one of the standard sizes. If you enter a size then the value entered must be square (you can enter either a single value or a ‘sizexsize’).

For use over the Internet it may be more appropriate to disable this option, which would mean that smaller individual files will be created for each layer, thus optimising download time.

Please see the Raster Configuration page for further information.
