To add Data Files to a Dataset

  • Click on one of the datasets in the Project Tree.
  • Select Dataset | Add Data or Data | New to access the Select Data to Add dialogue.
  • Browse to the correct directory and then select the data files to add to the dataset. You can select multiple data files, using the Shift and Ctrl modifiers, to load many data files into the dataset in a single operation. Note that there is a limit to the number of data files that can be added in a single operation, since the file names are concatenated within the File name field, and this buffer will eventually become full. You can normally load at least two hundred data files in a single operation.
  • Press Open to add the data files to the dataset.

The ordering of data files within the dataset determines the order in which the data files will be rendered. Data files at the top of the list within the dataset will be rendered first, and therefore will be overlaid with data files lower down in the list.

Moving, Copying and Deleting Data Files

  • To re-order the data files within a dataset, simply click on a data file and drag it to its new position within the dataset.
  • You can also move a data file from one dataset to another (in a different layer if desired) by dragging it into the appropriate position in the new dataset. This operation will be allowed only if the target dataset is of the same type as the source dataset
  • To copy a data file, hold down the Ctrl key whilst dragging the data file. You can copy to a new dataset or within the same dataset.
  • To delete a data file, right-click on the data file and select Delete from the popup menu. Confirm the deletion to remove the data file from the Project Tree.
  • You can also move, copy and delete data files using the Cut/Paste, Copy/Paste and Delete options on the Edit menu.
