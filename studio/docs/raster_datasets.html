Raster Datasets
Vector Datasets

Raster Datasets allow the loading of raster data, via an import filter, into MapLink Pro Studio. MapLink Pro Studio currently supports the following raster data formats:

  • Common raster formats

  • ARCS

  • ADRG

  • ArcGrid

  • ASCII DEM

  • ASRP

  • BSB Nautical Chart Format

  • CADRG

  • CIB

  • CRP

  • DBDBV

  • DMED

  • DTED

  • ECRG

  • GeoTIFF

  • Geospatial PDF

  • MrSID

  • NITF 2.1

  • USRP

  • Other Raster

Raster performance can be optimised through Raster Configuration.
