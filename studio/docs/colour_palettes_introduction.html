Colour Palettes - Introduction

MapLink Pro Studio currently offers a choice of four standard colour palettes:

  • Web-Safe palette (the default palette)
  • HSV palette
  • Pastel palette
  • Artistic palette

Each of the palettes displays a tool-tip which lists the; colour index, RGB value and the hexidecimal colour value of the colour cell under the mouse.

The colour palettes can be used to select a colour or to create a new user defined colour. You may also create a completely new palette.

The HSV, Pastel and Artistic palettes have space for up to 40 user-defined colours. The Web-Safe palette can contain over a 100 user defined colours.

You can import palettes from one of the standard palettes or from any Feature Book.

You can save palettes containing user defined colours.
