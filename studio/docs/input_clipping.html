Input Clipping

This allows the user to clip in the input coordinate space. The bottom left and top right coordinates of the clipping rectangle need to be defined in input coordinate system units.

Input Clipping is a property of a Dataset.

To define the clip boundaries for Input Clipping

  • Select the required Dataset
  • Use Dataset | Input Clipping... to open the input clipping page
  • Enter the bottom left and top right coordinates for the clipping rectangle
  • PressOK to accept
  • Reload the layer, if automatic data loading is not enabled.
