File Streaming
File Streaming Options

These options allow the modification of parameters that determine how a Vector Filter file is streamed in. The File Size parameter determines the maximum size of file, in megabytes, to be read in and processed in its entirety, i.e. into a contiguous area of memory. Any files larger than this size shall be read in and processed in pages.

The Page Size parameter determines the size, in megabytes, of the page or block that the file is streamed in.

Setting both entries to zero resets the file streaming behaviour to the default.
