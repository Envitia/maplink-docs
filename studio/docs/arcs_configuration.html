ARCS Configuration

Arcs Charts are an 8 bit colour depth format. When configuring the raster options the depth should be left as blank or set to 8. When projecting the data the options ‘Maintain palette when projecting palettised images’ and ‘Disable dithering’ should be selected as these options should ensure the quality of the map and reduce the disk space used.

Updates: Select the location of the Chart Updates.

Plan Indices: Select the Plans to ignore or load.

Border Masking: Select this option to mask out the outer part of the base plan.
