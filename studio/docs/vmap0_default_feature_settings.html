VMAP0 Default Feature Settings

The default settings for VMAP0 feature subclassing and text feature creation have been selected to make the best use of those VMAP0 attributes that can make a significant difference to the visual effect and the usefulness of map data created using VMAP0. The list below shows those features for which either or both of these options are used in the default version of vpffield.ini. The columns Sub 1 and Sub 2 give the attributes that are used to create the first and second level subclasses respectively; the Text column gives the attributes that is used for the creation of the corresponding text feature.
