Vector Datasets

Vector Datasets allow the loading of vector data, via an import filter, into MapLink Pro Studio. MapLink Pro Studio currently supports the following vector data formats:

  • ASCII

  • DXF

  • DAFIF

  • DBDBV

  • File Geodatabase (FileGDB)

  • GML2/GML3 Simple Features

  • Jeppesen Navdata

  • KML Simple Features

  • NITF 2.1

  • GDF For further information on this format, click here. Support for GDF is available as an add-on to the standard MapLink Pro configuration.

  • MIF. Support for MIF is available as an add-on to the standard MapLink Pro configuration.

  • NTF for Ordnance Survey BaseData.GB, Meridian, Land-Line Plus and OSCAR data. For further information on these formats, click here. Support for NTF is available as an add-on to the standard MapLink configuration

  • OpenStreetMap

  • OS MasterMap

  • OS VectorMap Local

  • OS Boundary Line 2000

  • S-57 (ENC and AML)

  • SHP

  • U.S. Census TIGER/Line

  • VPF for VMAP0 data.

  • Other Vector
