Multithreading Options

These options allow the user to request MapLink Pro Studio to use specified threading options. Multiple threads can significantly improve performance of loading and processing raster data but can also increase the memory requirements. Therefore these options are turned off by default.

Multithreading may not always be used if set, dependent on whether a process supports multiple threads.

Currently, multithreading can be applied to the following processes:

  • Raster Projection
  • Raster Pyramiding
