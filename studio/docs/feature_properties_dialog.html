Feature Properties Dialog

To open the Feature Properties dialog, ensure that the Feature Book is open

  • Select a Feature Class from the Feature Tree.
  • Open the Feature Properties dialog by using one of the following methods.
    • Press the Properties button the Feature Book toolbar.
    • Select Feature | Properties from the Feature Book menubar
    • Right click on the Feature Class and select Properties from the context menu

The Feature Properties dialog will automatically update to display the current Rendering Properties for the selected Feature Classes.

If multiple Feature Classes are selected then the common shared properties are displayed. For properties that are not common, such as each selected Feature Class having a different edge colour, the Feature Properties dialog will show a blank display.
