Toolbar Appearance

You may select from small, medium and large toolbars. Medium and large toolbars are particularly useful when using MapLink Pro Studio in a high-resolution mode (such as 1280x1024 or 1600x1200). By default the toolbar buttons are shown in the flat style, but you may prefer to use the older raised button style.
