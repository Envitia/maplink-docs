Creating a New Project Template

Any existing MapLink Pro Studio project can be used as a project template. The easiest way to construct a new template is to create a new project using the default template shipped with MapLink Pro. In this project define any settings that all projects based on this template should have. This is not just limited to the MapLink Pro Studio project settings; it is also possible to define layers, datasets and even source data that should be included in the template as well as any relevant Feature Book entries.

Additionally, should the resultant maps of projects based on this template be intended for use by systems using older versions of MapLink Pro the default output format for maps can be set in Project | Properties. This allows the default output format for maps to be set to a particular version of MapLink Pro.

For the default output map version to persist the template must be saved for MapLink Pro version 6 or newer.

If the project is saved when you create a Map then the version it is saved as will match the version of the Map. If the map version selected is older then MapLink Pro 6 then the Studio project will not contain the output map version selected.
