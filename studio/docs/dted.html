DTED

Digital Terrain Elevation Data. This is a uniform matrix of terrain elevation values. DTED Level 0 elevation post spacing is 30 arc second (nominally one kilometre). In addition to this discrete elevation file, a separate binary file provides the minimum, maximum, and mean elevation values computed in 30 arc second square areas (organised by one degree cells). DTED Level 0 also contains the NGA Digital Mean Elevation Data (DMED) providing minimum, maximum, and mean elevation values and standard deviation for each 15 minute by 15 minute area in a one degree cell. This can be interpreted as contour vector or raster. DTED data provides relief information as well as terrain in 3D and allows for calculations such as line of sight to be worked out.

Specification Reference: MIL-PRF-89020A.
