Fill Styles

MapLink Pro provides facilities for many different Fill styles. For example: simple Solid and Hollow fills, standard hatch and cross-hatch styles, large custom patterns and translucent and complex raster mode fills.

Note: Translucency is not supported on Windows NT 4.0 or Windows 95 - users will have to be running Windows 98, 2000 or XP and newer to take advantage of these features (not all Windows Operating Systems mentioned here are supported). Performance is heavily dependent upon the graphics hardware available when rendering with a translucent fill style. Testing on a small map initially is recommended until the desired style is decided upon.
