VMAP0 Data Organisation

The VMAP0 dataset structure is fully defined by the following documents:

  • Military Specification: Vector Smart Map (VMap) Level 0, NIMA, MIL-V-89039, 9/2/1995
  • Interface Standard for Vector Product Format, NIMA, MIL-STD-2407, 28/6/96.

It is not necessary to have a full understanding of the VMAP0 internal structures in order to use the dataset within MapLink Pro, but it is helpful to have a basic understanding of some of the more important structures involved.

As with other VPF datasets, VMAP0 is organised into Libraries, Coverages, Feature Classes and Features.

The top-level structure is the Library. Each VMAP0 CD-ROM will contain two libraries:

  • A Reference Library, called Rference.
  • A Data Library, using the name of the geographical coverage (e.g. EURNASIA).

For normal mapping purposes, only the thematic coverages contained in the Data Library are of interest.
