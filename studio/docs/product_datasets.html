Product Datasets

Product Datasets are preconfigured datasets stored in MapLink Pro Studio template files. They are intended to simplify the configuration and rendering of common data products, such as S57, VPF, OSMasterMap, OS VectorMap Local, etc.

These datasets have had any necessary subclassing already setup and are supplied with an associated Feature Book Section with all the rendering styles defined.

Note:

These datasets have been setup with a pre-determined TMC per MU value that the User should ensure is the same as that set for the project that the dataset is being added to.

The User also needs to ensure that the project and the dataset both use the same Palette.

Adding a Product Dataset

To add a Product Dataset to a Layer:

  • Click on one of the layers in the Project Tree.
  • Select Layer | Add Product Dataset to access the Product Datasets dialogue. This dialog is populated by the contents of the configuration file MapLinkStudioDatasets.ini, located in the Installation\config directory.
  • Select one of the available Product Datasets from the list and press Ok. Note: selecting one of the options will display a longer description of the dataset.
